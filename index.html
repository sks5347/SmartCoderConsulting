Filenet Migration

Solution provides reliable and lightning fast migration from Filenet to UCM. Features include flexible scheduling according to business while maintaining the content integrity and business continuity. This solution will migrate the document content, metadata, security, folders and can be custom tailored to business requirements. Migration involves mapping of metadata, security markings(security groups, roles, accounts), creating file plans and index areas.

Adobe Analytics/Omniture integration

Solution integrates WCC with Adobe analytics/omniture system. Integration allows an organization to enable/generate hundreds of reports for statistics. Solution is configurable and can be used to tracked any service/request in content server and also exposes a custom service which can be used by any external application using content server as content source to track content access statistics.

Integration with Artifactory, Version Control & Jenkin

Solution helps to integrate and perform common operation for building and deploying applications whose contents are maintained in WCC system. Solution adds capability:

  • To integrate with Jfrog artifatory and support Add and Delete artifact operation.
  • To integrate with Perforce version control to checkout, update and checkin deployment related file.
  • To integrate with Jenkin for triggering build and deployment of a application.

Workflow Content Migration

Custom workflow utility can migrate WCC workflow content from one system to another system without loosing any workflow data. Utility odes not require to disable workflow to move the content from source to target system.
